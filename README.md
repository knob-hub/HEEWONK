> 2026년 7월 4일 토요일 오후 2시 · 더테라스 웨딩홀

## 기술 스택

- Vite + React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion

## 로컬 실행

```sh
npm install
npm run dev
```

개발 서버: http://localhost:8080

## 빌드

```sh
npm run build
```

빌드 결과물은 `dist/` 폴더에 생성됩니다.

## 배포

Vercel을 통해 자동 배포됩니다.
